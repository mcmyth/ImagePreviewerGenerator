# ImagePreviewerGenerator

根据文件名数字排序并自动生成序列化预览图

**文件名必须包含序列数字!**

#### 配置项

* 页面宽度
* Express服务端监听端口

* 单页图像总数
* 单页图像列数
* 文件名标签字体大小

### 开始使用

* 在终端中执行命令`yarn`安装依赖
* 在`config.js`配置相关细则
* 在终端中执行命令`yarn reset`初始化目录
* 在`image/`中放入图像文件
* 在终端中执行命令`yarn start`
* 在`output/`中获取输出图像
